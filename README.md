# QTL Mapper  :mag:
QTL Mapper is tool that is written in `C#` to test and verify if a QTL region is present in the genetic map :scroll: by checking from genotype and phenotype of an organism :bug:\
<br/>
Architecture: `MVP` \
<br/>
Framework: `Winforms`\
